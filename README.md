# Vendr Scraper

## Опис проєкту
Vendr Scraper — це Python-скрипт для збору інформації про продукти з трьох категорій: **DevOps**, **IT Infrastructure**, **Data Analytics**.  
Дані збираються у вигляді словників з полями `name`, `category`, `price_range` та `description` і записуються у PostgreSQL базу даних.

Проєкт реалізований із використанням принципів **OOP**, **SOLID**, **DRY** та **KISS**, з багатопоточністю та чергами для ефективного скрейпінгу.

---

## Функціональність

- Збір продуктів із заданих категорій
- Використання **мокованих даних** для тестування
- Підтримка багатопоточності через `ScraperWorker`
- Запис даних у PostgreSQL через `DbWriterThread`
- Конфігурація через `.env` (User-Agent, таймаути, параметри БД)
- Тестовий режим для виводу продуктів у консоль без запису в БД

---

## Технології

- Python 3
- `requests` та `lxml` (для реального скрейпінгу)
- `psycopg2` для роботи з PostgreSQL
- `threading` і `queue` для багатопоточності
- `python-dotenv` для роботи зі змінними оточення

---

## Використання
Повний режим (запис у PostgreSQL):
``python vendr_scraper.py``


Після запуску всі продукти будуть збережені у таблиці products бази vendr_db.
